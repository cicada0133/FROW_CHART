# 🚀 Руководство по локальному запуску (Quick Start Guide)

Это руководство поможет вам развернуть генератор блок-схем C++ на локальном компьютере. Проект состоит из двух частей: **Backend (Python)** для парсинга кода и генерации схем, и **Frontend (React)** для предпросмотра в браузере.

---

## 📋 Требования (Prerequisites)
Перед началом убедитесь, что у вас установлены:
1.  **Python 3.10+** (проверьте командой `python --version`).
2.  **Node.js 18+** и **npm** (проверьте командами `node -v` и `npm -v`).
3.  **Git** (для клонирования проекта, если нужно).

---

## 🛠 Шаг 1: Настройка Backend (Python)
Откройте терминал в корневой папке проекта (`FROW_CHART`) и выполните:

1.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
2.  **Запустите сервер:**
    ```bash
    python server.py
    ```
    *Сервер запустится по адресу `http://localhost:8005`. Не закрывайте это окно терминала.*

---

## 🎨 Шаг 2: Настройка Frontend (Web Viewer)
Откройте **второе** окно терминала и перейдите в папку веб-интерфейса:

1.  **Перейдите в папку:**
    ```bash
    cd web_viewer
    ```
2.  **Установите Node-зависимости (только при первом запуске):**
    ```bash
    npm install
    ```
3.  **Запустите приложение в режиме разработки:**
    ```bash
    npm run dev
    ```
    *После запуска в консоли появится адрес (обычно `http://localhost:5173`). Перейдите по нему в браузере.*

---

## 🕹 Как пользоваться (Usage)

### Через веб-интерфейс (Рекомендуется)
1. Вставьте свой C++ код в левое поле редактора.
2. Справа мгновенно появится блок-схема.
3. Используйте панель настроек (Settings), чтобы изменить:
   - **Vertical Spacing:** Расстояние между блоками.
   - **Switch Case Style:** "Staggered" (компактно в шахматном порядке) или "Horizontal".
   - **Font Size:** Размер шрифта внутри блоков.
4. Нажмите **Export**, чтобы сохранить результат в формате PNG.

### Через консоль (CLI)
Если вам достаточно просто сгенерировать файлы по старинке:
1. Поместите код в файл `kod.txt` в корне проекта.
2. Запустите скрипт:
   ```bash
   python main_vertical.py
   ```
3. Результаты появятся в папке `flowcharts_kod`.

## ⚠️ Решение проблем (Troubleshooting)

- **Ошибка "Port 8005 is already in use":** Это значит, что предыдущий процесс сервера не закрылся. Закройте его вручную или используйте команду `taskkill /F /IM python.exe` (в Windows).
- **Схема не отображается:** Проверьте терминал с `server.py`. Если там красные ошибки, возможно, в C++ коде есть синтаксические ошибки, которые мешают парсеру.
- **Node.js не ставится:** Скачайте последнюю версию LTS с официального сайта [nodejs.org](https://nodejs.org/).
