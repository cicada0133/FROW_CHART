# C++ to GOST Flowchart Generator

Утилита для автоматической генерации блок-схем по исходному коду на языке C++. Блок-схемы строятся максимально близко к стандарту **ГОСТ 19.701-90**, что делает инструмент отличным помощником для оформления курсовых и лабораторных работ.

⚠️ **От автора:** Данный проект я делала для автоматизации рутинного построения блок-схем по ГОСТу. Генератор лучше всего подходит для небольших программ и алгоритмов. Сложные абстракции (классы, шаблоны) могут обрабатываться с упрощением.

## ✨ Новые возможности (v2.0)
- **ГОСТ-циклы For:** Полноценная поддержка блока `Preparation` (шестиугольник) для циклов `for`.
- **Интеллектуальный Switch:** Выбор между компактным "шахматным" порядком и классическим горизонтальным.
- **Улучшенная маршрутизация:** Автоматическое разнесение линий возврата и входа для предотвращения наслоений.
- **Interactive Web Viewer:** Полноценный веб-интерфейс для мгновенного предпросмотра схем с настройками в реальном времени.

## 🚀 Как пользоваться

### 1. Веб-интерфейс (Рекомендуется)
Самый удобный способ: запустить сервер и работать в браузере с живым превью.

**Запуск бэкенда:**
```bash
python server.py
```
**Запуск фронтенда:**
```bash
cd web_viewer
npm install
npm run dev
```
Откройте `http://localhost:5173` — там вы сможете вставлять код и сразу видеть результат, меняя отступы и стили "на лету".

### 2. Консольный запуск (`main_vertical.py`)
Генерирует схемы компактно по вертикали (режим по умолчанию). 
```bash
python main_vertical.py
```
Читает код из `kod.txt` и сохраняет в `flowcharts_kod`.

```bash
# Базовые библиотеки для рендеринга и парсинга
pip install schemdraw tree-sitter tree-sitter-cpp matplotlib

# Для работы веб-интерфейса
pip install fastapi uvicorn
```

## 🔴 Известные особенности
- При экстремальной вложенности (7+) может потребоваться ручная настройка `Vertical Spacing` в веб-интерфейсе.
- Сложные `switch` с `fallthrough` поведением (без `break`) рисуются как обычные блоки.
